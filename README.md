#### Usage

```sh
  hcat [FILE] [KEYWORD_OF_CSV]

  # hcat sample.txt abc,xyz
```
or
```sh
  [STRING] | hcat [KEYWORD_OF_CSV]

  # cat sample.txt | hcat abc,xyz
```

#### Install

`hcat` is copied in PATH.
